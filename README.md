# wc-ssl-journal

## 项目简介

`wc-ssl-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面记录了与 SSL/TLS 证书、Web 安全配置或相关网络状态相关的信息。

本仓库不针对任何单个域名或具体网站，而是作为一个通用的页面集合，用于存放和展示各类与证书、加密、安全检测相关的静态 HTML 内容。

## 目录说明

仓库根目录下包含以下内容：

- `index.html`：首页入口，提供页面导航或汇总链接。
- `pages/`：存放各个独立 HTML 页面的子目录。每个页面均有独立文件名，内容互不干扰。
- `assets/`：存放页面所需的静态资源文件，如 CSS 样式、JavaScript 脚本、图片等。
- `docs/`（可选）：存放额外的说明文档或参考材料。

目录结构示例：

```
wc-ssl-journal/
├── index.html
├── pages/
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/
│   ├── style.css
│   └── script.js
└── README.md
```

## 页面归档说明

- 每个 HTML 页面均为独立、完整的文档，可直接在浏览器中打开查看。
- 页面内容涉及 SSL/TLS 相关主题，包括但不限于证书信息、加密协议分析、安全检测结果等。
- 页面之间无依赖关系，可单独更新或添加，不影响其他页面。
- 仓库不提供在线服务，仅作为静态页面存档使用。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或对已有页面进行修正。
- 新增页面请放入 `pages/` 目录，并确保文件名唯一、无特殊字符。
- 若需更新首页 `index.html`，请同步维护导航链接。
- 本仓库不存储任何敏感信息（如私钥、密码等），请勿在页面中包含此类内容。
- 所有页面内容应保持客观、中立，不包含任何营销或诱导性表述。

## 其他

- 本仓库仅用于技术存档与展示，不提供任何形式的服务保证。
- 如有疑问或建议，欢迎提交 Issue 进行讨论。
